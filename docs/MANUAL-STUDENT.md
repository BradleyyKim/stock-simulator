# 학생 사용 매뉴얼

> 모의투자 시뮬레이션 학생용 안내서
> 최종 업데이트: 2026-03-26

---

## 1. 접속 및 로그인

1. 선생님이 알려준 URL에 접속합니다
2. **이름**을 드롭다운에서 선택합니다
3. **비밀번호** 4자리를 입력합니다
4. "로그인" 버튼을 누릅니다

> 비밀번호를 모르겠다면 선생님에게 문의하세요.

---

## 2. 화면 구성

### 상단 헤더

| 항목 | 설명 |
|------|------|
| **내 이름** | 현재 로그인한 학생 이름 |
| **보유 현금** | 현재 사용 가능한 현금 (매수/정보 구매에 사용) |
| **상태 뱃지** | 현재 게임 상태 표시 |

상태 뱃지 의미:
- `R3 거래 가능` → 3라운드 진행 중, 매수/매도/정보 구매 가능
- `R3 대기 중` → 다음 라운드 대기 중, 거래 불가
- `R3 결과 확인` → 라운드 결과 확인 중, 거래 불가

### 하단 탭 (4개)

| 탭 | 기능 | 언제 사용 |
|----|------|----------|
| **종목** | 전체 종목 현재가, 등락률 확인 + 매수/매도 | 거래 시 |
| **거래소** | 유료 정보 구매 (A/B/C 등급) | 힌트 공개 후 추가 정보가 필요할 때 |
| **내 자산** | 보유 현금, 보유 종목, 자산 추이 그래프 | 수시 확인 |
| **순위** | 전체 학생 수익률 순위 | 라운드 종료 후 |

## 3. 종목 확인 및 거래

### 종목 목록 보기
- "종목" 탭에서 5개 종목의 현재가와 등락률을 확인할 수 있습니다
- 종목은 가상 종목(인공지능, 에너지 등) 또는 실제 종목(삼성전자, NAVER 등)일 수 있습니다
- 종목을 클릭하면 상세 화면이 열립니다

### 종목 상세 화면
- **캔들스틱 차트**: 라운드별 가격 변동을 시각적으로 확인
- **현재가 / 등락률**: 직전 라운드 대비 변화
- **보유 수량**: 현재 이 종목을 몇 주 가지고 있는지

### 매수 (사기)
1. 종목 상세 화면에서 **"매수"** 버튼을 선택합니다
2. 수량을 입력합니다 (+ / - 버튼 또는 직접 입력)
3. 화면에 표시되는 **총 금액**을 확인합니다
4. **"매수"** 버튼을 눌러 주문합니다
5. 보유 현금에서 해당 금액이 차감됩니다

### 매도 (팔기)
1. 종목 상세 화면에서 **"매도"** 버튼을 선택합니다
2. 팔고 싶은 수량을 입력합니다
3. **"매도"** 버튼을 눌러 주문합니다
4. 보유 현금에 판매 금액이 추가됩니다

### 거래 규칙
- 거래는 **"거래 가능"** 상태에서만 할 수 있습니다
- 보유 현금보다 비싼 매수는 불가능합니다
- 보유 수량보다 많이 매도할 수 없습니다
- 가격은 **라운드 종료 시** 변동됩니다 (거래 중에는 가격이 바뀌지 않습니다)
- 공매도(보유하지 않은 주식 팔기)는 불가능합니다

## 4. 뉴스 (힌트) 확인

라운드가 시작되면 선생님이 뉴스를 순서대로 공개합니다. 뉴스는 **무료**이며 모든 학생에게 동시에 표시됩니다.

| 단계 | 라벨 | 특징 |
|------|------|------|
| 1단계 | **속보** (황색) | 모호한 정보, 방향만 짐작 가능 |
| 2단계 | **추가 보도** (주황) | 범위가 좁혀짐, 판단 근거 생김 |
| 3단계 | **긴급 속보** (빨강) | 핵심 정보 공개, 최종 판단 가능 |

> **팁**: 1단계에서 성급하게 매수하지 마세요. 2~3단계 뉴스를 기다렸다가 판단하는 것이 유리합니다. 하지만 너무 늦으면 이미 가격이 변한 후일 수 있습니다!

라운드 종료 후에는 **"뉴스 정답 분석"**이 표시되어 어떤 판단이 맞았는지 확인할 수 있습니다.

## 5. 정보 거래소

### 거래소란?
무료 뉴스(힌트)와 별도로, **유료 정보**를 구매할 수 있는 공간입니다. 좋은 정보일수록 비싸지만, 올바른 투자 판단에 도움이 됩니다.

### 사용 방법
1. 하단 **"거래소"** 탭을 누릅니다
2. 6개의 정보 카드가 표시됩니다 (구매 전에는 내용이 `???`로 숨겨져 있습니다)
3. 구매하고 싶은 카드를 클릭합니다
4. 가격을 확인하고 **"구매"**를 누릅니다
5. 보유 현금에서 차감되고, 정보 내용이 공개됩니다

### 정보 등급

| 등급 | 카드 색상 | 개수 | 가격 (기본) | 특징 |
|------|----------|------|-----------|------|
| **C등급** | 회색 | 3개 | 1,000원 | 일반적인 정보, 누구나 알만한 내용 |
| **B등급** | 파란색 | 2개 | 3,000원 | 괜찮은 정보, 방향성 힌트 |
| **A등급** | 금색 | 1개 | 5,000원 | 핵심 정보, 구체적인 내용 |

> 가격은 선생님이 조정할 수 있으므로 차시마다 다를 수 있습니다.

### 주의사항
- **거래 시간에만** 구매할 수 있습니다 (대기 중/결과 확인 상태에서는 불가)
- 같은 정보를 **중복 구매할 수 없습니다** (이미 구매한 카드는 비활성화)
- 구매한 정보는 **"내 정보 목록"**에서 언제든 다시 볼 수 있습니다
- 정보 구매에 현금을 너무 많이 쓰면 투자할 돈이 줄어듭니다!
- 모든 정보를 살 필요는 없습니다 — **선택적으로** 구매하세요

## 6. 내 자산 확인

### 자산 요약
- **총 자산**: 보유 현금 + 보유 종목의 현재 평가액
- **보유 현금** (초록): 매수/정보 구매에 사용 가능한 금액
- **투자 금액** (파랑): 현재 보유 중인 종목들의 평가액 합계

### 자산 추이 그래프
- 라운드별 내 총 자산의 변화를 그래프로 확인할 수 있습니다
- 기준선(시작 금액)과 비교하여 수익/손실을 파악하세요
- 2라운드 이상 진행되어야 그래프가 표시됩니다

### 보유 종목
- 보유 중인 종목의 이름, 수량, 현재가, 평가액이 표시됩니다
- 0주인 종목은 표시되지 않습니다

---

## 7. 순위 확인

- **수익률** 기준으로 전체 학생의 순위가 정렬됩니다
- 수익률 = (현재 총자산 - 시작 자금) / 시작 자금 x 100
- 내 이름은 **파란색**으로 하이라이트됩니다
- 1/2/3위는 메달 표시가 됩니다
- 선생님이 순위판을 비공개로 설정한 경우 **"비공개"** 메시지가 표시됩니다

---

## 8. 용돈 시스템

- 매 라운드가 시작될 때 **용돈**(기본 100,000원)이 자동 지급됩니다
- 투자에 실패하여 현금이 부족해도 다음 라운드에서 다시 참여할 수 있습니다
- 용돈 금액은 선생님이 설정하며, 차시에 따라 다를 수 있습니다
- 차시가 바뀌어도 이전 자산이 이어질 수 있습니다 (선생님 설정에 따라)

## 9. 게임 흐름 예시 (한 라운드)

```
1. [대기 중] 선생님이 라운드를 시작할 때까지 기다립니다
       ↓
2. [거래 가능] 팝업: "라운드 3 시작!"
   → 용돈이 자동 지급됩니다
   → 이전 라운드 결과를 참고하여 전략을 세웁니다
       ↓
3. [속보 공개] 팝업: "힌트 1단계 공개!"
   → 모호한 뉴스를 읽고 어떤 종목에 영향이 있을지 생각합니다
       ↓
4. (선택) 거래소에서 추가 정보를 구매합니다
   → 무료 뉴스만으로 부족하면 유료 정보 구매를 고려합니다
       ↓
5. [추가 보도 공개] 팝업: "힌트 2단계 공개!"
   → 판단이 더 명확해집니다. 매수/매도를 시작합니다
       ↓
6. [긴급 속보 공개] 팝업: "힌트 3단계 공개!"
   → 핵심 정보 확인 후 최종 매수/매도를 합니다
       ↓
7. [거래 마감] 팝업: "거래 마감"
   → 주가가 변동됩니다. 더 이상 거래할 수 없습니다
   → "뉴스 정답 분석"을 읽고 자신의 판단을 되돌아봅니다
       ↓
8. [대기 중] 다음 라운드 준비
```

## 10. 투자 팁

| 팁 | 설명 |
|----|------|
| **분산 투자** | 한 종목에 전부 넣지 마세요. 여러 종목에 나눠 투자하면 리스크가 줄어듭니다 |
| **뉴스 분석** | 뉴스가 어떤 산업에 영향을 주는지 생각하세요. 직접 관련이 없는 종목도 간접 영향을 받을 수 있습니다 |
| **타이밍** | 좋은 뉴스가 나온 후에 사는 것보다, 힌트를 분석해서 미리 사는 게 더 큰 수익을 줍니다 |
| **현금도 전략** | 모든 돈을 투자할 필요는 없습니다. 하락이 예상되면 현금을 보유하세요 |
| **정보 구매 판단** | 모든 정보를 사면 투자금이 줄어듭니다. 필요한 정보만 선택적으로 구매하세요 |
| **손절매** | 이미 산 종목이 하락할 것 같으면, 더 떨어지기 전에 파는 것도 전략입니다 |
| **뉴스의 함정** | 2단계까지의 뉴스는 의도적으로 모호하게 작성되어 있습니다. 성급한 판단을 조심하세요 |

## 11. 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| 로그인이 안됨 | 이름과 비밀번호를 다시 확인하세요. 선생님에게 문의하세요 |
| 매수/매도 버튼이 안 눌림 | "거래 가능" 상태인지 확인하세요. 상단 뱃지를 확인합니다 |
| "현금이 부족합니다" | 보유 현금보다 비싼 매수는 불가. 수량을 줄이거나 다른 종목을 매도하세요 |
| "보유 수량이 부족합니다" | 가지고 있는 것보다 많이 팔 수 없습니다. 수량을 확인하세요 |
| 거래소에서 정보 구매가 안됨 | 거래 시간에만 구매 가능합니다. 현금이 충분한지 확인하세요 |
| 이미 구매한 정보 카드가 클릭 안됨 | 같은 정보는 중복 구매할 수 없습니다 |
| 순위가 안 보임 | 선생님이 순위판을 비공개 설정했을 수 있습니다 |
| 화면이 안 바뀜 / 멈춤 | 인터넷 연결 확인 후 새로고침(F5) 하세요 |
| 팝업 알림이 안 보였음 | 팝업은 약 2.5초 후 자동으로 사라집니다. 상단 뱃지로 현재 상태를 확인하세요 |
