# 관리자(교사) 사용 매뉴얼

> 모의투자 시뮬레이션 관리자용 안내서
> 최종 업데이트: 2026-03-26

---

## 1. 접속 및 인증

1. 브라우저에서 `사이트주소/admin`으로 접속합니다
2. 관리자 비밀번호를 입력합니다
3. "입장하기"를 클릭합니다

---

## 2. 관리자 화면 구성

### 상단 헤더
- **차시 / 라운드 정보**: 현재 진행 상황 표시 (예: 차시 1 | 라운드 3/6)
- **학생 등록** 버튼: 학생 일괄 등록
- **학생 화면** 버튼: 학생 화면으로 이동하여 학생 시점에서 확인

### 3개 탭

| 탭 | 용도 | 사용 시점 |
|----|------|----------|
| **운영** | 수업 중 게임 진행 (라운드 시작/마감, 힌트 공개, 순위판, 학생 현황) | 수업 중 |
| **콘텐츠** | 시나리오별 라운드 콘텐츠 입력 (뉴스, 힌트, 정보 거래소, 변동률) | 수업 전 준비 |
| **설정** | 게임 설정 + 시나리오 관리 + 실제 주식 데이터 시나리오 생성 | 최초 설정 |

## 3. 수업 전 준비

### 3.1 학생 등록

1. 상단 "학생 등록" 버튼을 클릭합니다
2. 텍스트 영역에 학생 정보를 입력합니다 (한 줄에 하나씩):
   ```
   홍길동, 1234
   김철수, 5678
   이영희, 9012
   ```
3. "학생 등록하기"를 클릭합니다

> 비밀번호를 생략하면 기본 `0000`이 설정됩니다.

### 3.2 게임 설정 (설정 탭)

| 항목 | 기본값 | 설명 |
|------|--------|------|
| 시작 자금 | 100,000원 | 학생 초기 보유 현금 |
| 라운드 용돈 | 100,000원 | 매 라운드 시작 시 전체 학생에게 자동 지급. 0이면 미지급 |
| C등급 정보 가격 | 1,000원 | 정보 거래소 C등급 구매 가격 |
| B등급 정보 가격 | 3,000원 | 정보 거래소 B등급 구매 가격 |
| A등급 정보 가격 | 5,000원 | 정보 거래소 A등급 구매 가격 |

설정 변경 후 반드시 **"설정 저장"**을 눌러주세요.

### 3.3 시나리오 준비

시나리오를 준비하는 방법은 **2가지**입니다:

#### 방법 A: 실제 주식 데이터로 자동 생성 (권장)

실제 한국 주식 과거 데이터를 가져와 변동률이 자동 적용된 시나리오를 만듭니다.

1. **설정 탭** → "실제 데이터로 시나리오 생성" 섹션으로 이동합니다

2. **종목 선택**: 기본 5개 종목이 설정되어 있습니다. 6자리 종목 코드를 변경할 수 있습니다

   | 카테고리 | 기본 종목코드 | 종목명 |
   |---------|-------------|--------|
   | 반도체 | 005930 | 삼성전자 |
   | 인공지능 | 035420 | NAVER |
   | 에너지 | 373220 | LG에너지솔루션 |
   | 바이오 | 207940 | 삼성바이오로직스 |
   | 게임 | 263750 | 펄어비스 |

   > 종목 코드는 네이버 금융(finance.naver.com)에서 종목명을 검색하면 확인할 수 있습니다.

3. **기간 입력**: 시작일과 종료일을 선택합니다
   - **1~2년 전 데이터**가 적합합니다 (학생이 결과를 모르도록)
   - 6라운드 기준 약 **6~8주** 분량이 적당합니다

4. **라운드 수**: 기간을 몇 개의 라운드로 나눌지 설정합니다 (기본 6)

5. **"데이터 가져오기"** 클릭
   - 실제 주가 데이터가 수집되어 라운드별 변동률이 표시됩니다
   - 상승은 빨간색(+), 하락은 파란색(-)으로 표시됩니다

6. **미리보기** 테이블에서 라운드별 변동률을 확인합니다

7. **"시나리오 생성"** 클릭 → 시나리오가 자동 등록됩니다

8. **콘텐츠 탭**으로 이동하여 뉴스/힌트를 입력합니다 (3.4 참고)

#### 방법 B: 직접 시나리오 작성

1. **설정 탭** → 시나리오 관리 → **"+ 새 시나리오"** 클릭
2. 시나리오 이름, 설명 입력
3. 라운드를 추가하고 각 라운드의 뉴스, 힌트, 변동률을 직접 입력
4. 저장

> 기본 시나리오 2개(기본 시장 체험, 변동성 장세)가 미리 등록되어 있습니다. "편집"을 눌러 수정하거나 그대로 사용할 수 있습니다.

### 3.4 콘텐츠 입력 (콘텐츠 탭)

시나리오의 라운드별 콘텐츠를 간편하게 입력하는 전용 화면입니다.

1. **시나리오**를 드롭다운에서 선택합니다
2. **라운드 번호**(R1, R2, ...)를 클릭합니다
3. 실제 데이터 시나리오인 경우, 상단에 **해당 라운드의 실제 기간**이 표시됩니다
4. **"관련 뉴스 찾기"** 버튼을 클릭하면 빅카인즈(BIG KINDS)에서 해당 기간의 실제 경제 뉴스를 검색할 수 있습니다
5. 검색된 뉴스를 참고하여 각 항목을 입력합니다:

| 항목 | 설명 | 입력 팁 |
|------|------|---------|
| **뉴스 헤드라인** | 라운드 종료 시 발표되는 뉴스 제목 | 실제 기사 제목을 참고 |
| **1단계 (속보)** | 모호한 첫 번째 힌트 | 산업/종목을 특정하기 어렵게 |
| **2단계 (추가 보도)** | 범위를 좁히는 두 번째 힌트 | 방향성은 알 수 있지만 확신은 못 하게 |
| **3단계 (긴급 속보)** | 핵심 정보를 담은 세 번째 힌트 | 구체적이지만 너무 노골적이지 않게 |
| **결과 분석** | 라운드 종료 후 학생에게 보여줄 해설 | 어떤 판단이 맞았는지, 교육 포인트 |
| **종목별 변동률** | 각 종목의 가격 변동 퍼센트 | 실제 데이터 시나리오면 자동 입력됨 |
| **정보 거래소 아이템** | C등급 3개 + B등급 2개 + A등급 1개 | 등급별 정보 품질 차등화 |

6. **"라운드 N 저장"**을 눌러 저장합니다
7. 다음 라운드로 이동하여 반복합니다

> **변동률 입력**: 소수점 입력이 가능합니다 (예: -2.8, +5.3). 실제 데이터 시나리오에서는 변동률이 자동으로 채워져 있으므로 수정하지 않아도 됩니다.

## 4. 수업 중 운영 (운영 탭)

### 4.1 시나리오 선택 및 시작

1. **운영 탭**의 "시나리오 선택"에서 사용할 시나리오를 클릭합니다
2. **"이전 자산 이어서 시작"** 체크 여부를 결정합니다:
   - **체크 안 함**: 전원 시작 자금으로 리셋 (첫 차시 또는 경연대회)
   - **체크**: 이전 차시의 보유 현금을 유지 (연속 수업)
3. 시나리오를 클릭하면 종목 가격이 초기화됩니다

### 4.2 라운드 진행

#### 1단계: 라운드 시작
- **"라운드 N 시작"** 버튼을 클릭합니다
- 학생들에게 "라운드 시작!" 팝업이 표시됩니다
- **용돈이 자동으로 지급**됩니다 (설정된 금액만큼)

#### 2단계: 힌트 순차 공개
- **"힌트 1단계 공개"** 클릭 → 학생에게 "속보" 표시
- (학생들이 뉴스를 분석하고 토론하는 시간)
- **"힌트 2단계 공개"** 클릭 → "추가 보도" 표시
- (학생들이 매수/매도를 시작, 정보 거래소에서 정보 구매)
- **"힌트 3단계 공개"** 클릭 → "긴급 속보" 표시
- (학생들이 최종 판단)

#### 3단계: 거래 마감
- **"거래 마감 및 결과 반영"** 클릭
- 시나리오의 변동률이 주가에 적용됩니다
- 전체 학생의 자산이 재계산됩니다
- 학생 화면에 **"뉴스 정답 분석"**이 표시됩니다

#### 4단계: 다음 라운드 준비
- 결과 토론 시간을 가집니다
- **"다음 라운드 준비"** 클릭
- 다시 1단계로 돌아갑니다

### 4.3 라운드 제어 요약

```
[대기 중]
  │
  ├─ "라운드 N 시작" 클릭 (용돈 자동 지급)
  │
  ▼
[거래 진행 중]
  │
  ├─ "힌트 1단계 공개" → 학생에게 속보 표시
  ├─ "힌트 2단계 공개" → 학생에게 추가 보도 표시
  ├─ "힌트 3단계 공개" → 학생에게 긴급 속보 표시
  │
  │   (학생들은 이 사이에 종목 거래 + 정보 거래소 구매)
  │
  ├─ "거래 마감 및 결과 반영" → 주가 변동, 자산 재계산, 분석 표시
  │
  ▼
[결과 확인]
  │
  ├─ "다음 라운드 준비" 클릭
  │
  ▼
[대기 중] (반복)
```

### 4.4 순위판 관리

- **"순위판 공개"** / **"순위판 비공개"** 토글로 학생 화면의 순위판 표시 여부를 조절합니다
- 교육 목적에 따라 순위를 숨기거나 공개할 수 있습니다
- 경연대회 모드에서는 최종 라운드까지 비공개 후 공개하면 효과적입니다

### 4.5 학생 현황 확인

운영 탭 우측에 전체 학생의 현황이 실시간으로 표시됩니다:
- 순위, 이름, 보유현금, 총자산, 수익률(%), 접속 상태(초록 점)

### 4.6 라운드 초기화 (긴급 시)

- 거래 진행 중 문제가 발생하면 **"라운드 초기화"** 버튼을 사용합니다
- 현재 라운드를 취소하고 이전 상태로 되돌립니다
- 학생 화면의 뉴스도 모두 사라집니다
- 확인 팝업이 표시되므로 실수로 누르는 것을 방지합니다

## 5. 수업 진행 타임라인 (20분/라운드)

```
시간     관리자 행동                        학생 화면
──────────────────────────────────────────────────

0분     "라운드 N 시작" 클릭              → "거래 가능" + 용돈 지급

4분     "힌트 1단계 공개"                 → 속보(황색) 표시
        학생들 뉴스 분석 + 토론 유도         학생들 종목 분석

8분     "힌트 2단계 공개"                 → 추가 보도(주황) 표시
        매수/매도 시작 유도                 거래 활발 + 정보 거래소 구매

12분    "힌트 3단계 공개"                 → 긴급 속보(빨강) 표시
        최종 거래 안내                     마지막 매수/매도

16분    "거래 마감 및 결과 반영"           → 주가 변동 + 결과 분석 표시
        뉴스 정답 해설                     자산 변화 확인

18분    결과 토론                          순위 확인 (공개 시)

20분    "다음 라운드 준비"                 → 대기 중
```

## 6. 정보 거래소 콘텐츠 작성 가이드

### 좋은 콘텐츠 작성 원칙

| 등급 | 정보의 성격 | 작성 예시 |
|------|-----------|----------|
| **C등급** | 누구나 아는 일반 상식 수준 | "최근 정부가 경제 정책을 검토 중이다" |
| **B등급** | 방향성은 알 수 있지만 구체적이지 않음 | "금융 정책 변화가 성장주에 유리할 수 있다" |
| **A등급** | 구체적인 산업/종목을 특정할 수 있는 정보 | "기준금리 인하로 AI, 반도체 종목이 수혜" |

### 작성 팁
- C등급으로만은 판단할 수 없어야 합니다
- B등급은 "이런 방향일 것 같다"는 느낌을 줘야 합니다
- A등급은 확실한 근거를 제공하되, 너무 노골적이면 교육 효과가 떨어집니다
- 가짜 정보(미스디렉션)를 일부 포함하면 정보 분별력 교육에 효과적입니다
- 실제 데이터 시나리오인 경우, **"관련 뉴스 찾기"** 버튼으로 실제 뉴스를 참고하세요

### 변동률 설정 가이드 (직접 작성 시나리오용)
- 뉴스 대상이 아닌 종목: **0% 또는 ±2% 이내**로 설정
- 일반적인 영향: **±3~8%**
- 큰 이벤트: **±10~20%**
- 한 라운드에 모든 종목이 같은 방향으로 움직이면 비현실적 → 종목별로 차별화
- 소수점 입력 가능 (예: -2.8, +5.3)

> 실제 데이터 시나리오에서는 변동률이 자동 적용되므로 직접 설정할 필요 없습니다.

## 7. 차시 간 운영 방법

### 연속 수업 (자산 누적)
1. 새 시나리오 선택 시 **"이전 자산 이어서 시작"** 체크
2. 학생의 보유 현금이 유지됩니다
3. 보유 종목은 새 시나리오의 종목과 다를 수 있으므로, 현금으로 정산됩니다

### 경연대회 모드 (리셋)
1. 체크박스를 **해제**합니다
2. 전원 동일한 시작 자금으로 리셋됩니다
3. 순위판을 최종 라운드까지 비공개하면 긴장감이 높아집니다

### 매 라운드 용돈
- 기본 100,000원이 라운드 시작마다 자동 지급됩니다
- 돈을 모두 잃은 학생도 계속 참여할 수 있습니다
- 설정 탭에서 금액을 변경하거나 0으로 설정하여 비활성화할 수 있습니다

## 8. 실제 주식 데이터 시나리오 — 전체 워크플로우

실제 데이터 기반 시나리오를 준비하는 전체 과정입니다.

```
[설정 탭]
  │
  ├─ 1. 종목 코드 확인/변경 (기본 5개 제공)
  ├─ 2. 기간 입력 (1~2년 전 데이터 권장)
  ├─ 3. "데이터 가져오기" 클릭
  ├─ 4. 미리보기에서 변동률 확인
  ├─ 5. "시나리오 생성" 클릭
  │
  ▼
[콘텐츠 탭]
  │
  ├─ 6. 생성된 시나리오 선택
  ├─ 7. R1 선택 → 기간 표시 확인
  ├─ 8. "관련 뉴스 찾기" 클릭 → 빅카인즈에서 실제 뉴스 확인
  ├─ 9. 뉴스를 참고하여 힌트 1/2/3 입력
  ├─ 10. 정보 거래소 아이템 6개 입력
  ├─ 11. "라운드 1 저장"
  ├─ 12. R2, R3, ... 반복
  │
  ▼
[운영 탭]
  │
  ├─ 13. 시나리오 선택 → 수업 시작
  └─ 14. 라운드 진행 (4장 참고)
```

### 핵심 포인트
- **변동률은 100% 실제 시장 데이터** — 주관적 판단이 들어가지 않습니다
- **뉴스/힌트는 해당 기간의 실제 경제 뉴스** — 빅카인즈에서 검색하여 참고
- 관리자가 할 일은 **뉴스 고르기 + 힌트 텍스트 작성 + 정보 거래소 콘텐츠 작성**뿐

## 9. 문제 해결

| 문제 | 원인 | 해결 |
|------|------|------|
| 학생이 로그인 안됨 | PIN 불일치 또는 미등록 | "학생 등록"에서 재등록 |
| 거래가 안됨 | phase가 'trading'이 아님 | "라운드 시작" 버튼 확인 |
| 가격이 안 변함 | 거래 마감 안 함 | "거래 마감 및 결과 반영" 클릭 |
| 순위판이 안 보임 | 순위판 비공개 상태 | 순위판 공개 토글 |
| 뉴스/힌트가 안 보임 | 시나리오에 콘텐츠 미입력 | "콘텐츠" 탭에서 입력 후 저장 |
| 정보 거래소가 비어있음 | 라운드에 정보 아이템 미입력 | "콘텐츠" 탭에서 아이템 입력 |
| 시나리오가 없음 | 최초 실행 시 자동 생성됨 | 새로고침 또는 설정 탭에서 확인 |
| 실제 데이터 가져오기 실패 | 종목 코드 오류 또는 네트워크 | 6자리 코드 확인, 인터넷 확인 |
| "관련 뉴스 찾기"가 안 보임 | 직접 작성 시나리오 | 실제 데이터 시나리오에서만 표시됨 |
| 화면이 안 바뀜 | Firebase 연결 문제 | 와이파이 확인, 새로고침 |
| 데이터 초기화하고 싶음 | - | 시나리오 재선택 (이어서 시작 해제) |
| 용돈이 지급되지 않음 | roundAllowance가 0 | 설정 탭에서 용돈 금액 확인 |
